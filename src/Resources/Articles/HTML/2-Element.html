First and foremost every HTML document starts with one special tage called the doctype tag, here's what it looks like!


  1. <!DOCTYPE html>

The purpose of the tag is to let the browser know this is a HTML document it also lets the browser know what version of HTML we are using.


You will notice that the name of the tag is enclosed in <>. All tags are contained within arrow brackets and further down the document require a similar closing tag.

For out doc type tag this looks like this...


  1. </html>

Let's look at some other tags and see what they can do!


  1. <!DOCTYPE html>: //This line tells the browser that we're using the latest version of HTML.

  1. <html>: //This is the main container for the entire web page.
  1. <head>: //Inside the head, we put important information about the page, like its title (shown on the browser tab).
  3. <title>: //This tag sets the title of the web page, which appears in the browser tab.
  5. <body>: //Inside the body, we put all the visible content of the web page.
  7. <h1>: //This tag creates a big heading.
  9. <p>: //This tag creates a paragraph of text.
  11. <img>: //This tag adds an image to the page.
  13. <a>: //This tag creates a link to another web page.
