Internal Styling

Get ready to take your web pages to the next level with CSS internal stylesheets! With internal stylesheets, you can add fantastic styles to your web creations without cluttering your HTML. It's like having a secret notebook where you can jot down all your design ideas!



So, what exactly is a CSS internal stylesheet? Well, you already know that HTML is the foundation of a web page, like the framework of a house. CSS is the creative part that adds colors, fonts, and other cool styles to make your website look awesome. And an internal stylesheet is a special section within your HTML file where you can write all your CSS code.



Now, let's dive into how you can start using CSS internal stylesheets. It's really exciting! Inside your HTML file, you need to add a special tag called <style>. This tag acts as a container for your CSS code. You place it between the <head> and </head> tags, just like this:

  1. <!DOCTYPE html>
  2. <html>
  3. <head>
  4. <title>My First Web Page</title>
  5. </head>
  6. <body>
  7. <h1>Welcome to my Web Page!</h1>
  8. <p>This is a paragraph of text.</p>
  9. <img src="image.jpg" alt="My Image">
  10. <a href="https://www.example.com">Click here to visit Example.com</a>
  11. </body>
  12. </html>

Within the <style> tags, you can unleash your creativity and write CSS code to style your web page. You can change colors, fonts, sizes, and so much more! It's like having a magical pen to bring your design ideas to life.


For example, if you want to make all the paragraphs on your web page have a red color, you can add the following CSS code within the <style> tags:

  1. <style>
  2. p {
  3. color: red;
  4. }
  5. </style>

See how the CSS code is written inside the <style> tags? In this case, we're targeting all the <p> elements (paragraphs) and saying that they should have a red color. And just like that, you've added a touch of style to your web page!


One cool thing about internal stylesheets is that you can apply different styles to different elements on the same page. You can target specific elements by using their HTML tags, like <h1> for headings or <a> for links. It's like having a superpower that lets you customize each part of your web page individually!
