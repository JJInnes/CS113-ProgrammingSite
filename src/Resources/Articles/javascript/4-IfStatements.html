I’d bet by now you are beginning to see how programs are built. You put data in variables, then use operators to do the basic arithmetic with them. But how do we make decisions?


This is where conditional statements, or if statements come in. They allow us to decide whether or not to run blocks of code based on logical expressions, and are written like this:

  1. if(){
  3. }

The condition for the if statement goes into the "()" and a block of code goes between the curly brackets, this code will only run if the condition evaluates to true.


For example the code block here will run:

  1. if(0 < 1){
  3. }

But the code block here will not:

  1. if(0 > 1){
  3. }

And this code block will only run if the variable “x” is a positive number

  1. if(x > 0){
  3. }

Along with if statements, you may also use else statements.


ELSE statements take no condition but have their own code block, which only runs if the previous IF statement does not run its code (as its condition was false), for example:

  1. if(x < 0){
  2.   console.log("Negative");
  3. } else {
  4.   console.log("Positive");
  5. }

ELSE IF statements are good if you want to chain conditions together, for example:

  1. if(x < 0){
  2.   console.log("Negative");
  3. } else if (x == 0) {
  4.   console.log("Zero");
  5. } else {
  6.   console.log("Positive");
  7. }

This new condition is only checked if the previous one is false. You can chain many else if statements together.
