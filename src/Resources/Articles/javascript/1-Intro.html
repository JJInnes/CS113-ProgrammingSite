Javascript allows you to add client-side scripting to your web page, providing you a means to create dynamic behaviour affecting your website's components.


With javascript you can improve user interaction and the functionality of your page by manipulating your pages HTML and CSS dynamically in your scripts. You can add behaviour to buttons, forms and much more. It even allows you to create complex animations, games and visuals that would not otherwise be possible.


This series of articles will cover the absolute basics of using Javascript, giving you the tools you need to get experimenting and learning independently.
